# GeoWay

Planejador de missões fotogramétricas para drone, com terrain-following e exportação
para KMZ no formato DJI WPML (compatível com DJI Fly). Rodando 100% no navegador,
em um único arquivo HTML/JavaScript — sem instalação, sem backend.

**Versão atual:** 1.1

## Uso pretendido

Ferramenta de apoio ao planejamento de voos de mapeamento aéreo (REURB, georreferenciamento
rural, levantamentos topográficos), com malha de fotogrametria e ajuste de altitude por
elevação do terreno (SRTM). O uso da ferramenta é livre para qualquer pessoa; o
código-fonte, no entanto, é fechado — veja a seção de direitos autorais abaixo.

⚠️ **Aviso de segurança:** esta é uma ferramenta de planejamento, não um sistema
certificado de gerenciamento de tráfego aéreo. Todo arquivo KMZ gerado deve ser
conferido pelo operador (relevo real, limites legais de altura, distâncias de RTH,
compatibilidade do modelo de drone) antes de qualquer voo. O autor não se
responsabiliza por missões executadas a partir dos arquivos exportados.
